Django commands

django-admin commands

Command Description
django-admin --version Displays django version
django-admin help Displays usage information and a list of the commands provided by each application
django-admin help --commands Displays a list of all available commands.
django-admin help <command> Displays a description of the given command and a list of its available options.
django-admin startproject my_project Creates django project with my_project name

python manage.py commands

Command Description
python manage.py startapp my_app Creates django application with my_app name
python manage.py runserver Runs the django project
python manage.py migrate Applyies all migrations
python manage.py makemigrations my_app ...
python manage.py shell Opens python shell
python manage.py createsuperuser Allows to crreate super user
